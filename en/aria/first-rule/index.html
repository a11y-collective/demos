First rule of ARIA

Div with class that'll trigger alert:

div class="js-alert"
Show alert

Add CSS:

div class="js-alert button"
Show alert

Add role="button":

div class="js-alert button"
role="button"
Show alert

Add tabindex=0:

div class="js-alert button"
role="button"
tabindex="0"
Show alert

And if you have a toggle button you also can add aria-pressed to tell a screen reader it's a toggle.

See for an example of aria-pressed MDN ARIA: button role.

Native HTML button and onclick:

button class="js-alert"
