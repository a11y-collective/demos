Link an SVG

Link to a Twitter account, using an SVG as icon

Preferred option: add text

Add the text Twitter inside the link and hide svg from the accessibility tree with aria-hidden="true".

Follow us on Twitter

Second option: use screen-reader-text

Use a span with the class="screen-reader-text" on the text "Follow us on Twitter" inside the link, to hide from vision, but announced by a screen reader.

Follow us on Twitter

Third option: use an aria-label

Use the aria-label="Follow us on Twitter" on the link.

Fourth option: add the accessible name to the SVG

Give the SVG a title, an aria-labelledby and a role="img"
