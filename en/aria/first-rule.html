First rule of ARIA

Div with onclick:

div onclick="showAlert()"
Show alert

Add CSS:

div onclick="showAlert()" 
class="button"
Show alert

Add role="button":

div onclick="showAlert()" 
class="button"
role="button"
Show alert

Add tabindex=0:

div onclick="showAlert()" 
class="button"
role="button"
tabindex="0"
Show alert
div onclick="showAlert()" 
class="button"
role="button"
tabindex="0"
onkeypress="showAlert()
Show alert

And if you have a toggle button you also can add aria-pressed to tell a screen reader it's a toggle.

See for an example of aria-pressed MDN ARIA: button role.

Native HTML button and onclick:

button onclick="showAlert()"
