Landmarks

Landmarks give semantic structure to a web page. Bonus: Screen readers can navigate on it.

Examples landmarks

Important: Only one <main> per page, preferable containing the main content.

  • <header>
  • <nav>
  • <main>
  • <section>
  • <aside>
  • <footer>

Additional roles for landmarks

  • <header>
  • <header role="banner">
  • <form>
  • <form role="search">

Label landmarks that aren't unique on a page

Give them an aria-label.

  • <nav aria-label="Main">
  • <nav aria-label="Social media">
  • <aside aria-label="Contact information">
  • <aside aria-label="Related articles">

Older browsers

For Internet Explorer 10 and older:

  • <nav role="navigation">
  • <main role="main">
  • <aside role="complementary">
