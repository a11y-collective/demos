About buttons and links

Recreate a button with a div

Div with onclick:

div onclick="showAlert()"
Show alert

Add CSS:

div
onclick = "showAlert()"
class   = "button"
Show alert

Add role="button":

div
onclick = "showAlert()"
class   = "button"
role    = "button"
Show alert

Add tabindex=0:

div
role     = "button"
onclick  = "showAlert()"
class    = "button"
tabindex = "0"
Show alert
div
onclick    = "showAlert()"
class      = "button"
role       = "button"
tabindex   = "0"
onkeypress = "showAlert()
Show alert

And if you have a toggle button you also can add aria-pressed to tell a screen reader it's a toggle.

See for an example of aria-pressed MDN ARIA: button role.

Recreate a link with a span

Span with onclick:

span onclick="location.href='http://a11y-collective.com';"

a11y-collective.com

Add some extra CSS to make it look like a link.

span
class = "dummy-link"

The A11Y Collective

Add an onkeypress event, role=link, tabindex=0, some extra CSS:

span
role       = "link"
tabindex   = "0"
class      = "link"
onclick    = "location.href='http://a11y-collective.com'"
onkeypress = "location.href='http://a11y-collective.com'"
The A11Y Collective
